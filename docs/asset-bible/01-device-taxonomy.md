# 01 — Таксономия устройств (общая для всех рас)

Функциональный состав набора один для всех рас (в духе оригинальной Ascendancy:
двигатели, генераторы, щиты, оружие, сканеры, спец-устройства). Раса меняет
только визуальный язык, не состав.

## Архетипы

| # | Device Type | База формы (архетип) | Read при взгляде |
|---|---|---|---|
| 0 | `engine` | корпус + сопло/эмиттер тяги сзади | «толкает» |
| 1 | `generator` | компактное ядро + радиаторы/контур | «излучает энергию» |
| 2 | `shield` | эмиттер + дуга/кольцо проекции | «оборачивает» |
| 3 | `weapon_beam` | ствол/фокусатор вдоль оси | «целится» |
| 4 | `weapon_launcher` | кассета/барабан ячеек | «выпускает залпом» |
| 5 | `scanner` | тарелка/антенна/сенсорная гроздь | «слушает» |
| 6 | `special` | нестандартный силуэт вне рядов выше | «непонятно и интересно» |

## Правила читаемости (для всех рас)

1. **Силуэт решает.** Тип устройства должен угадываться по чёрному силуэту,
   до материалов и деталей.
2. **Один акцент.** У каждого устройства ровно одна светящаяся/цветная зона —
   функциональное «сердце» (сопло, эмиттер, ядро). Цвет акцента — расовый.
3. **Тех-уровни.** Внутри типа устройства прогресс技术 передаётся параметром
   `Size Class` + плотностью деталей, а не сменой силуэта: игрок должен видеть
   «тот же двигатель, но круче».
4. **Иконка = модель.** UI-иконки рендерятся из тех же GeoNodes-моделей
   (Detail Level = 1.0, орто-камера) — никакого отдельного 2D-арта, который
   разойдётся с моделью.
