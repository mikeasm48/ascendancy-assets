> ✅ Сверено с фактическими ассетами (2026-07-08). В репозитории ассетов раса
> лежит в папке **`bionics`**:
> `~/.ascendancy/assets/races/bionics/ships/shipyard_constructor.glb` (56 мешей, Sketchfab)

> биокерамика, плавные объёмы, без швов» для кораблей подтвердилась по форме,
> но не по цвету: корпуса не бежево-фиолетовые, а глянцево-белые, почти
> монохромные. Здания — не органика, а чистый «кремово-белый хайтек»
> с серебром, золотом и стеклом.
>

## 1. Стилевая формула

Полированный био-минимализм: глянцево-белые бесшовные монолиты непрерывной
кривизны — техника как выращенный фарфоровый организм; во флоте формы капель,
вилок и линз с тёмными врезками-«зевами», в архитектуре — радиальные башни из
насаженных на ось дисков и куполов с серебряными, золотыми и стеклянными поясами.

## 2. Геометрический язык

### Корабли (факт: shipyard_constructor.glb)

| Параметр | Значение |
|---|---|
| Базовые формы | капли, линзы, блюдца (`Saucer`), стручки-шаттлы (`Shuttle 1–4`), корпуса-«камертоны» — U/Y-развилки (`Hull 1–11`, `Conjunction`), обтекатели-«ножны» (`Sheath 1–3`), несущие «скелеты» (`Carcass 1–5`), ажурная сфера-решётка (`Sphere.001` — воронои-паттерн) |
| Углы и фаски | углов нет — непрерывная кривизна, smooth-шейдинг (121k вершин на 234k треугольников — сглаженная органика, ср. с гранёными Humans) |
| Симметрия | билатеральная у корпусов-развилок, радиальная у блюдец и сферы; корабли плоские, «камбалой»: высота ≪ длины и ширины |
| Панелизация | нет; членение — утопленные тёмные кольца и щели-«жабры» на белом монолите (см. baseColor-текстуру) |
| Гриблы | классических гриблов нет; допустимы: усики-антенны (`Antenna`), утопленные турели (`Laser/Missile/Gauss/Wave cannon` — тоже сглаженные капли), рёбра-«крылья» (`Wing 1–7`) |
| Запрещено | прямые углы, плоские плиты, болты/трубы, видимые швы и крепёж |

Один материал на весь набор (`Base`, 100% поверхности) — единый «фарфоровый»
корпус, никакого мультиматериального членения: форма важнее раскраски.

### Здания (факт: building_constructor.glb)

| Параметр | Значение |
|---|---|
| Базовые формы | 14 семейств `Building_01…14` (с вариациями — 31 здание): цилиндрические башни из стопок дисков/тарелок, блюдца на ножке, купола, кольца-террасы; в плане почти всё — окружность |
| Углы и фаски | скругления и тела вращения; жёсткие рёбра только на технологических поясах |
| Симметрия | радиальная (осевая) — доминирующий признак застройки |
| Панелизация | горизонтальные пояса материалов (бетон/серебро/золото/стекло) вместо плит |
| Гриблы | шпили-иглы на вершинах, тонкие ножки-опоры; мелкой детализации нет |
| Запрещено | сеточные кварталы «коробками» (это язык Humans), видимые швы |

## 3. Палитра

### Корабли — фактический материал GLB

| Роль | HEX | Metal/Rough | Источник |
|---|---|---|---|
| Корпус основной (глянцевый белый) | `#FCFCFC`–`#E4E4E4` (~82% пикселей) | 1.0 / карта MR (глянец) | материал `Base`, baseColor 1024² |
| Полутон/затенённые зоны | `#CCCCCC`, `#B4B4B4`, `#9C9C9C` | — | там же |
| Врезки, кольца, «зевы» | `#3C3C3C`–`#545454` | — | там же |
| Насыщенные акценты | **отсутствуют** (0 пикселей с sat>0.35) | — | там же |

Флот Shuffie принципиально ахроматичен: белый глянец + тёмно-графитовые
углубления. Цветовой акцент, если нужен геймплейно (team color, свечение
двигателей), добавлять эмиссией в углубления — в исходнике его нет.

### Здания — фактические материалы GLB (доля поверхности в треугольниках)

| Роль | HEX | Metal/Rough | Материал в GLB | Доля |
|---|---|---|---|---|
| Корпус основной (белый «бетон») | `#E7E7E7` | 0 / 0.3 | `Concrete_01` | 56.7% |
| Серебряные пояса | `#E7E7E7` | 1.0 / 0.2 | `Silver_01` | 16.4% |
| Золотые акценты | `#F0C37D` | 1.0 / 0.2 | `Gold_01` | 10.6% |
| Остекление | `#7183AC` | 1.0 / 0.2 | `Glass_01` | 9.5% |
| Тёмно-серые врезки | `#5E6470` | 0 / 0.3 | `Gray_01` | 6.7% |

Правило пропорций: ~57% матовый белый, ~26% полированный металл
(серебро+золото), ~10% стекло, ~7% тёмный контраст.

### Сводная палитра для процедурных материалов

| Роль | HEX | Материал |
|---|---|---|
| Корпус основной | `#F2F2F2` (глянцевый белый; корабли rough≈0.1–0.3) | `MAT_shuffie_hull` |
| Корпус вторичный | `#E7E7E7` metal (серебро, здания) | `MAT_shuffie_hull_b` |
| Акцент (свечение/team color) | в исходниках нет; резерв — в тёмные врезки | `MAT_shuffie_emissive` |
| Детали/врезки | `#3C3C3C`–`#5E6470` графит + `#F0C37D` золото (только здания) | `MAT_shuffie_detail` |

## 4. Интерфейс `GN_Style_Shuffie`

Стандартный контракт (`00-pipeline.md`): Geometry / Seed / Detail / Accent Col.
`Accent Col` в стиле Shuffie применять только к эмиссии в углублениях
(двигатели, турели) — на корпус цвет не выносить, иначе ломается ахроматичный
«фарфор» исходников.

## 5. Реализация кита

- Сглаживание базы: Subdivision + Smooth by `part_id` (сохранить читаемость зон) —
  подтверждено: исходники полностью smooth-шейденные, форма монолитная.
- `GN_Style_Shuffie_Skin` — вместо чешуи/пор из черновика фактический приём
  исходников: **утопленные тёмные кольца и щели** — inset выбранных «поясов»
  по нормали внутрь + тёмный графитовый материал в углублении; амплитуда × Detail.
- `GN_Style_Shuffie_Nodes` — органические наросты на `greeble_slot`-гранях:
  инстансы капель разного масштаба, слияние через Mesh Boolean (Union) на высоком
  Detail. По исходникам наросты — редкие и крупные (антенна, турель-капля),
  не «сыпь» мелких узлов.
- Для development-силуэтов кораблей: развилка/вилка корпуса (U/Y-split вдоль
  продольной оси) — самый узнаваемый приём набора `Hull 1–11`.
- Для зданий: генератор «стопки тел вращения» (диск/купол/кольцо по профильной
  кривой) + распределение поясов материалов сверху вниз.

## 5b. Объединённый набор зданий v2 (2026-07-11)

`tools/build_shuffie_building_set.py` собирает
`bionics/buildings/building_constructor_v2.glb` (55 узлов) из трёх
исходников — текущий сет НЕ перезаписывается:

- **A** — этот сет (`bionics/buildings/building_constructor.glb`, 31 здание):
  металл/стекло, радиальные башни — основа доминант;
- **B** — бионический city-set, ранее временно у Humans
  (`humans/buildings/building_constructor.glb`, 23 меша): инопланетные
  формы и текстуры (белая керамика + бирюза + лайм), мелкие групповые
  модели большой площади;
- **C** — `refs/buildings/Shuffie/SpaceDock_shuffie_1.glb`,
  `SpaceShield_shuffie_1.glb`: орбитальные док и щит (форма как есть).

Гармонизация под флот (ахроматичный белый глянец + графит):

| Материал A | Было | Стало |
|---|---|---|
| `Concrete_01` | белый матовый | белый фарфор, rough 0.25 |
| `Silver_01` | серебро | без изменений |
| `Gold_01` | золото | **бирюзовый металл** `#29999E`-зона (мост к текстурам B) |
| `Glass_01` | синее стекло | бирюзовое стекло |
| `Gray_01` | серый | графит врезок флота |

Текстуры B сохранены (бирюза/лайм — теперь фирменный акцент застройки),
добавлен фарфоровый глянец (rough 0.22). У щита C оранжевая линза
перекрашена в бирюзу (hue-shift Base Color + Emission текстур). Док C —
серебристый металл.

Маппинг доминант: 9 из A (COLONY_BASE=02_18, FACTORY=03_24, FARM=02.002,
LABORATORY=09_25, RESEARCH_CAMPUS=05_30, MEGAFACTORY=07.001,
HYDROPONIFIER=08_17, METROPLEX=14_20, POWER_PLANT=10_22) + 5 из B
(OUTPOST=Mushroom, HABITAT=Skyscraper 5, SKY_NET=Droadcast center,
ECO_BOOSTER=Ringed building, TERRAFORMING=Skyscraper 4). Пропсы (8): из B —
Random blocks 1/2, Social housing, Highway, Canopy, Spire, Facility; из A —
Pylon. Орбитальные: SPACE_DOCK, SPACE_SHIELD (уровни — скейлом в игре;
лазеры/фазеры Shuffie — TODO). Остальные 31 группа обоих сетов сохранены
узлами `VARIANT_*` (kind=variant) — богатство вариантов не потеряно.

Extras узлов: `building_id`, `display_name`, `kind`
(building/prop/orbital/variant), `level`, `style="shuffie"`, `source`.
Контрольный лист: `renders/preview/approval_buildings_shuffie_v2.png`.

Запуск: `blender -b -P tools/build_shuffie_building_set.py`
