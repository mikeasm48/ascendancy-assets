# Раса: Humans — стилевой кит

> ✅ Сверено с фактическими ассетами (2026-07-08):
> `~/.ascendancy/assets/races/humans/ships/shipyard_constructor.glb` (160 модульных частей, Blender I/O v5.1.19)
> `~/.ascendancy/assets/races/humans/buildings/building_constructor.glb` (23 меша, Sketchfab).
> Значения палитр извлечены из материалов/текстур GLB. Прежний черновик
> («индустриальный функционализм») подтвердился для кораблей, но НЕ для зданий —
> у Humans два визуальных регистра, см. ниже.

## 1. Стилевая формула

Два регистра одной цивилизации:

- **Флот («верфь»)** — сварной модульный металл: серые боксовые корпуса с фасками
  и скосами, плотный крупный грибл (баки, вентрешётки, фермы, мачты, тарелки),
  техника как инструмент, собранный из каталога частей.
- **Город («колония»)** — ретрофутуристичная белая керамика: плавные сужающиеся
  шпили, грибовидные башни и купола над прямоугольной жилой застройкой; панели
  бирюзовые, акцентные канты — лаймовые.

Общее ядро: модульность и явная «сделанность» — и корабль, и город собраны из
читаемых функциональных блоков; никакой органики, ничего «выращенного».

## 2. Геометрический язык

### Корабли (факт: shipyard_constructor.glb)

| Параметр | Значение |
|---|---|
| Базовые формы | боксы и цилиндры, вытянутые вдоль продольной оси (Z в GLB); типичная часть ~1.5×2×5 |
| Углы и фаски | 90° + скосы/фаски ~45° (части так и названы: `hull_angle`, `angle_Bar`, `anglePort`, `eng_body_slant`); шейдинг жёсткий, гранёный (split normals: 472k вершин на 357k треугольников) |
| Симметрия | билатеральная; сборки длинные, «поездом» (см. `ScoutShip`, `BattleShip`, `CruiserShip` внутри файла) |
| Панелизация | средние прямоугольные плиты, оконные блоки (`Window_Block`, `Dual_Window`), видимые швы |
| Гриблы | баки (`Tank`), вентблоки (`Vent_Block`), фермы (`Trusswork-Wing`), мачты и антенны (`Instrument_Mast`, `5 mast`), тарелки (`Dish_Panel`), хардпоинты оружия (`hardpoint_*` — 15+ вариантов) |
| Запрещено | органические кривые, асимметрия корпуса, «бесшовный» монолит |

Каталог из 160 частей делится на семейства: `hull_*` (корпуса), `eng_*`
(двигатели: `eng_trumpet`, `eng_onion`, `eng_5Pole`…), `Bridge_*` (мостики),
`hardpoint_*` (базы/стволы оружия), крылья/фермы, инструменты. Сборка кораблей —
JSON из `startingfleet/` ссылается на части как `shipyard_constructor#<index>`.

### Здания (факт: building_constructor.glb)

| Параметр | Значение |
|---|---|
| Базовые формы | сужающиеся шпили с юбками-кольцами (`Spire`), грибовидные башни (`Mushroom-like building`), купола-навесы (`Canopy 1–3`), кольцевое здание (`Ringed building`), 6 типовых небоскрёбов + блочная застройка (`Random blocks`, `Social housing`) и хайвеи (`Highway section`) |
| Углы и фаски | доминанты города — непрерывная кривизна (smooth-шейдинг), низовая застройка — прямоугольные блоки |
| Симметрия | у башен радиальная/осевая, у кварталов — сеточная |
| Панелизация | криволинейная сеть скруглённых панелей: белый корпус + бирюзовые поля + лаймовые канты; швы дорисованы normal map |
| Гриблы | практически нет — читаемость на дальней дистанции (это city-set) |

## 3. Палитра

### Корабли — фактические материалы GLB (доля поверхности в треугольниках)

| Роль | HEX | Metal/Rough | Материал в GLB | Доля |
|---|---|---|---|---|
| Корпус основной | `#818184` | 0.48 / 0.8 | `Mat_Main` | 84.0% |
| Корпус вторичный | `#BFBFC2` | 0 / 0.7 | `Mat_Seco` | 5.2% |
| Металл светлый | `#E7E7E7` | 1.0 / 0.4 | `Mat_MetalL` | 5.6% |
| Металл тёмный | `#595959` | 1.0 / 0.4 | `Mat_MetalD` | 3.2% |
| Тёмные врезки | `#191919` | 0 / 1.0 | `Mat_Dark` | 0.8% |
| Свечение белое | `#E7FFF8` ×2 | — | `Glow_Whi` | 0.2% |
| Свечение синее | `#4B95FF` ×2 | — | `Glow_Blu` | 0.2% |
| Свечение оранжевое | `#FF6C27` ×4 | — | `Glow_Ora` | 0.4% |
| Стекло | `#89BFE7` | 1.0 / 0.7 | `Glass` | 0.3% |

Правило пропорций: ~85% нейтральный серый корпус, ~9% металлы, ≤1% суммарного
глоу. Оранжевый — самый яркий акцент (emissive strength 4), синий/белый — служебная
подсветка (strength 2).

## 5. Реализация кита

- `GN_Style_Humans_Panels` — разрезка граней по `panel_density`, inset + bevel,
  случайное лёгкое смещение плит по нормали (seed). Для domain=building —
  криволинейная панельная сеть (бирюзовые поля + лаймовый кант по границе панели).
- `GN_Style_Humans_Greebles` — инстансы из коллекции `GRB_humans` на грани с
  `greeble_slot`, ориентация по нормали, плотность × Detail. Коллекцию наполнять
  переиспользованием реальных частей из `shipyard_constructor.glb`: баки, вентблоки,
  мачты, тарелки, хардпоинты — стиль уже задан ими.
- Силуэт: фасочный деформер — срез углов базовой формы под 45°
  (подтверждено реальными частями `hull_angle` / `eng_body_slant`).
- Шейдинг: жёсткие грани (autosmooth с малым углом) — корабли Humans гранёные,
  не сглаженные.
