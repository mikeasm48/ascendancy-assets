## 4. Интерфейс нод-группы `GN_Style_<Race>`

Обязательные входы (одинаковы у всех китов — контракт из `00-pipeline.md`):

```
Geometry   (меш архетипа с атрибутами part_id / greeble_slot / panel_density)
Seed       (int)
Detail     (float 0–1)
Accent Col (color — по умолчанию из палитры выше)
```

Выход: `Geometry` с назначенными материалами.
